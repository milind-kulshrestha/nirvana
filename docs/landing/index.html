Nirvana

AI-powered stock watchlist tracker

A native desktop app for tracking your watchlists with live market data and a conversational AI assistant. Your data stays on your machine — private, fast, and always available.

 Download for macOS ⊞ Download for Windows

Features

Everything you need to track the market

AI Assistant

Chat with Claude about your portfolio, get instant analysis, and ask questions about any stock in your watchlist.

Live Market Data

Real-time quotes powered by OpenBB and Financial Modeling Prep. Prices, moving averages, and historical charts.

Local & Private

All data stored locally on your machine in SQLite. No cloud accounts, no tracking, no data leaving your computer.

Smart Caching

Background data refresh with DuckDB — charts and historical data load instantly after the first fetch.

Under the Hood

Built with modern tools

Tauri v2

Native performance, tiny bundle size. Rust-powered shell with system webview.

React + TailwindCSS

Fast, responsive frontend with shadcn/ui components and Zustand state management.

Python + FastAPI

Async backend handling market data, AI conversations, and local database operations.
