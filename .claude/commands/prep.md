# Project Context Snapshot

Load and maintain project context across sessions:

1. Read docs/PROJECT_STATUS.md for current state
2. Review docs/CHANGELOG.md for recent changes
3. Check docs/reference/ for codebase overview
4. Summarize:
   - Active milestones and progress
   - Recent changes and their impact
   - Next priority tasks
   - Any blockers or dependencies

Then ask: "What would you like to work on?"
